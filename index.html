Ques1. Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype 
// parent object
const parent ={
    name: "Parent",
    age:50,
    about:info,
}

// attaching method to parent
function info(){
    console.log(`${this.name} is ${this.age} years old`)
}

// children object
const child={
    name:"child",
    age:21,
   childAbout: Object.create(parent), 
}

child.childAbout.about.call(child);

explaination:

Object.create(parent) it create a new object. create a [[prototype]] which store the methods and properties of parent object in ChildAbout as a value an empty {}


output:

child is 21 years old

Ques3. Add a method to calculate sum of all elements in Array in array's protype, use that method to calculate sum for multiple arrays 
function sum(){
    return this.reduce((prev,current)=>prev+current);
 }
 const arr= Array();
 Object.setPrototypeOf(arr,sum);
 
 let arr1=[4,2,42,1];
 let arr2=[24,1,2,1];
 let arr3=[4,5,32,2];
 let arr4=[9,8,2,1];
 
 console.log(Object.getPrototypeOf(arr).call(arr1));

 console.log(Object.getPrototypeOf(arr).call(arr4));

arr is an array which has prototype propertie which stores a method sum() with the help of setPrototypeOf ( Object.setPrototypeOf(arr,sum)) and get that function with this Object.getPrototypeOf(arr) here we want to call the sum funtion which is expecting a this value which is provide by call() method explicitly.
